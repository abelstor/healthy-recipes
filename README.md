# Healthy Recipes

## This is a basic responsive website about healthy food recipes, built using HTML, CSS.

### Features:

- Responsive design that adapts to different screen sizes.
- Clean and modern layout with a focus on visual appeal.
- Easy navigation with a simple menu.

### Technologies Used:
- HTML5 for structuring the content.
- CSS3 for styling and layout.
- Media queries for responsiveness.
- Soon to be enhanced with JavaScript for interactivity.
